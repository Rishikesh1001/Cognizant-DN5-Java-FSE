# Exercise 1 - Singleton Pattern

## Objective
To implement the Singleton Design Pattern in Java, ensuring that only one instance of the Logger class is created throughout the application.

## Files
- Logger.java
- Main.java

## How to Run
1. Compile the Java files.
2. Run Main.java.
3. Verify that both objects are the same instance.

## Expected Output
Logger instance created successfully.
Both objects are the same: true
