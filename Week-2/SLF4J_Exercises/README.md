# SLF4J Exercises

This folder contains the SLF4J hands-on exercise for Week 2 of the Cognizant Digital Nurture 5.0 Java FSE program.

## Exercise 1 - Logging Error Messages and Warning Levels

### Objective
Learn how to use the SLF4J logging framework to log messages at different severity levels.

### Files Included
- LoggingExample.java
- pom.xml

### Technologies Used
- Java 21
- Maven
- SLF4J API
- SLF4J Simple

### Sample Output

```
INFO  - Application Started
WARN  - This is a warning message
ERROR - This is an error message
INFO  - Application Finished
```

### Outcome
Successfully implemented logging using the SLF4J framework with INFO, WARN, and ERROR log levels.
