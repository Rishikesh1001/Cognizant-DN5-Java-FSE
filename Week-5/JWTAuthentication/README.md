# JWT Authentication

## Objective
This project demonstrates JWT (JSON Web Token) generation using Spring Boot.

## Technologies Used
- Java 21
- Spring Boot 3
- Spring Web
- JJWT
- Maven

## Features
- Generate JWT token
- REST API endpoint

## Endpoint

GET /token?username=<username>

Example:

GET /token?username=rishi

Response:

A JWT token is generated.
